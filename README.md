# OSCE-Timer-GUI-ID
OSCE Timer with Python and Google Voice with Bahasa Indonesia

Aplikasi ini adalah modifikasi berbasis antarmuka (GUI) dan penambahan fitur Google Text-to-Speech bahasa Indonesia dari proyek asli versi CLI milik Quinexus.

This application is a modification of Quinexus's original CLI-based project, featuring a graphical user interface (GUI) and the addition of Indonesian Google Text-to-Speech capabilities.
